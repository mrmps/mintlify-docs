Welcome to the AgentSerp API documentation!

This API allows you to search, extract, summarize, rerank, classify, and chat with AI-powered endpoints.

- All endpoints require Bearer authentication.
- See each endpoint page for request/response details and examples.

</Note>

All API endpoints are authenticated using Bearer tokens. Include your API token in the Authorization header of each request.
