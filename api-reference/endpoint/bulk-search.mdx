---
title: 'Bulk Search'
openapi: 'POST /bulk/search'
---

Run multiple search queries in bulk.

<ParamFields />

## Example Request

```bash
curl -X POST https://api.agentserp.com/bulk/search \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"queries": ["openai", "chatgpt"]}'
```
