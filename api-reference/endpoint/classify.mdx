Classify texts into predefined labels.

<ParamFields />

## Example Request

```bash
curl -X POST https://api.agentserp.com/classify \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"texts": ["What is OpenAI?"], "labels": ["AI", "ML"]}'
```
